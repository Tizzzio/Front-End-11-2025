## Presentazione

_Benvenuto nel progetto di un E-commerce finto realizzato con HTML, CSS e JavaScript! Questo progetto simula un negozio online dove gli utenti possono visualizzare prodotti, aggiungerli al carrello e vedere il totale del loro acquisto. Puoi trovare il codice sorgente completo nei file `index.html`, `index.css` e `script.js`. All'interno del file `script.js`, i prodotti sono definiti in un array di oggetti, ognuno con proprietÃ  come `id`, `nome`, `prezzo` e `immagine`. Il file `index.html` struttura la pagina web, includendo sezioni per la visualizzazione dei prodotti e del carrello. Le immagini sono state generate automaticamente.
